
# iotcALB

`iotcALB` supports the Management Strategy Evaluation (MSE) process for
Indian Ocean albacore tuna (*Thunnus alalunga*), managed by the [Indian
Ocean Tuna Commission (IOTC)](https://www.iotc.org).

See the [IOTC Albacore MSE
homepage](https://iotcalb.bluematterscience.com/) and the [Trial
Specifications Document](https://iotcalb.bluematterscience.com/tsd) for
more details on the overall process and technical details respectively.

## Installation

``` r
# install.packages("pak")
pak::pak("Blue-Matter/iotcALB")
```
